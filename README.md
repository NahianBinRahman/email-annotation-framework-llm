# 🧠 Human Annotation Framework for Evaluating Professional Email Responses in LLM Systems

## 🎯 Objective
This project focuses on designing a structured annotation framework to evaluate AI-generated professional email responses.

The goal is to improve response quality, consistency, and usefulness through human-guided labeling.

---

## 🧩 Project Overview
- Created a labeled dataset of AI-generated email responses  
- Designed annotation guidelines for consistent evaluation  
- Evaluated responses based on multiple quality dimensions  
- Identified issues and provided reasoning for each case  

---

## 🏷️ Annotation Criteria
Each response was evaluated based on:

- Quality (High / Medium / Low)  
- Tone (Professional / Neutral / Informal)  
- Completeness  
- Clarity  
- Usefulness  

---

## ⚠️ Common Issues Identified
- Polite but not useful responses  
- Missing key details (timeline, next steps)  
- Vague or generic answers  
- Lack of professional tone  

---

## 📊 Dataset
- 50 real-world professional email scenarios  
- Includes:
  - AI-generated responses  
  - Quality labels  
  - Issue identification  
  - Detailed reasoning  

---

## 🔍 Key Insights
- High-quality responses include clear next steps and timelines  
- Tone alone is not enough — usefulness is critical  
- Many medium responses fail due to lack of actionable detail  
- Structured responses improve clarity and effectiveness
